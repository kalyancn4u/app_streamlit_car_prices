# 🚗 Car Price & Range Predictor

A machine learning web app that estimates used car prices in **Indian Rupees** (₹)

> 🛠️ For the engineering rationale and the bugs this version fixes, see
> [docs/DESIGN_NOTES.md](docs/DESIGN_NOTES.md).
>
> 🛡️ For a beginner-friendly explanation of how the app prevents wrong inputs
> (and what `__pycache__`, smoke tests, etc. are), see
> [docs/INPUT_VALIDATION_GUIDE.md](docs/INPUT_VALIDATION_GUIDE.md).
>
> 🤖 To automate testing, deployment, and retraining with GitHub Actions
> (CI / CD / CDD), see
> [docs/CI_CD_GITHUB_ACTIONS.md](docs/CI_CD_GITHUB_ACTIONS.md).

---

## 📖 Overview

- Inputs: Make, Model, KM driven, Age, Mileage, Engine, Fuel type, etc.
- Trains a **Random Forest Regressor** for exact price.
- Trains a **Random Forest Classifier** for price range (budget buckets).
- Interactive web UI with dropdowns and sliders for car specs.

A machine learning web app that estimates:

- **Exact selling price** (in ₹ Lakhs/Crores)
- **Price Range category** (Low / Medium / High) with the actual ₹ interval (e.g., ₹3–6 Lakhs)

## ⚡ Quick Start

### 1. Prerequisites

- **Python 3.9+** ([Download](https://www.python.org/downloads/))

### 2. Clone or navigate to the project folder

```bash
cd path/to/your/project
```

### 3. Set up a virtual environment (recommended)

**Using built-in `venv` (no extra install):**

```bash
python -m venv venv
```

- **Windows (CMD)**: `venv\Scripts\activate.bat`
- **Windows (PowerShell)**: `venv\Scripts\Activate.ps1`
- **macOS / Linux**: `source venv/bin/activate`

**Using Conda (optional):**

```bash
conda create -n car-price python=3.11 -y
conda activate car-price
```

### 4. Install dependencies (if not done already)

```bash
pip install -r requirements.txt
```

### 5. Train the model (first time only)

```bash
python train_model.py
```

> This trains both models and writes five files into `models/`:
> `price_model.pkl`, `range_model.pkl`, `feature_columns.json`,
> `range_config.json`, and `metadata.json`.

### 6. Run the web app

```bash
streamlit run app.py
```

Your browser will open at **http://localhost:8501**.

## 🧠 Usage

1. Select a **Make / Brand** — the **Model** dropdown updates to that brand's variants.
2. Choose **Fuel**, **Transmission**, **Seller type**, and **Seats**
   (every option, including *Dealer* and *CNG*, is available).
3. Adjust the **Age, KM driven, Mileage, Engine, and Max power** sliders.
4. Click **Estimate price**.
5. Read the **exact price** (₹ Lakhs/Crores) and the **budget band**
   (Low / Medium / High) with its rupee interval and gauge.

## 🔧 Troubleshooting

| Issue                         | Fix                                                                                  |
| :---------------------------- | :----------------------------------------------------------------------------------- |
| `ModuleNotFoundError`       | Activate your virtual environment and run `pip install -r requirements.txt` again. |
| `Model artifacts not found` | Run `python train_model.py` to generate the `models/` files.                     |
| Port 8501 is busy             | Use `streamlit run app.py --server.port 8502`.                                     |

## 📚 Background

- **Streamlit** converts Python scripts into web UIs without HTML/CSS.
- **scikit-learn** provides the Random Forest algorithm that learns pricing patterns.
- **joblib** saves the trained model to disk for fast reloading.

> **Note:** Predictions are estimates based on historical data—use as a guide, not a guaranteed sale price.

Built with [Streamlit](https://streamlit.io) and [scikit-learn](https://scikit-learn.org).
